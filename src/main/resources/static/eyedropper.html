How It Works

1 The Geometry Swap

When switched to this mode, the main white LED shoots light down onto a physical colored object placed on the Petri dish stage. The light bounces off the object, travels straight up, and hits the TCS3200 sensor hovering in the lid.

2 Data Normalization

The raw sensor DEC values are intercepted by our backend algorithm. A scaling function val = (raw / max) * 255 normalizes the inputs into the perfect 0–255 integer range standard for digital RGB displays.

3 Base-16 Transpiler

The backend architecture executes an instant transpilation of the three normalized integers into a standard 6-digit hexadecimal string prefixed with a '#'. This allows frictionless color-matching for CSS and HTML.

4 Calibration Database

The system queries a localized JSON lookup table that maps strict mathematical HEX codes to human-readable industrial names (e.g., "BIT Durg Charcoal Gray").
